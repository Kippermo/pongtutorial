Step 1: Creating a Canvas

First things first, we need a place to play! Javascript has a special object called a canvas, which is exactly what it sounds like: a canvas is a rectangular place on which you can draw things on a webpage.

var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");
function draw() {
ctx.clearRect(0, 0, canvas.width, canvas.height);
}
setInterval(draw, 10);







The first two lines in this block of code are somewhat confusing, so I'll break it down for you.

The first line gets the canvas object (which is created in the html) and sets it equal to a variable called canvas.

The next line takes that canvas's context and puts it into a variable called ctx. This context is the way that we will be able to draw to this canvas.

The next portion is very crucial. In line 3, we define a function called draw.

Draw is a special function that is called whenever a javascript file is run.

Outside of the draw function, we set the draw interval to 10, meaning that it will execute whatever is contained in the draw function every ten miliseconds.

As you will soon see, this repeated drawing allows for us to create animation and movement.

So far, the only thing that we are drawing is a clear rectangle. The rectangle starts at the point (0, 0) on the canvas (the upper left-hand corner) and has a width and height equal to that of the canvas. In other words, we are simply drawing a rectangle to cover the entire canvas.

This code should simply create a large, white rectangle on your screen.
